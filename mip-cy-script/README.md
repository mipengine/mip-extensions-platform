## 示例

### 固定位广告
在MIP HTML中,直接使用标签, 用于正常显示固定位广告。示例如下:

```html
   <section class="adwraper" id="topad_xxx"></section>
   <mip-dp-script ></mip-dp-script>
```

### 基本用法
```html
<mip-cy-script>
    自定义内容，可以嵌套其他组件
</mip-cy-script>
```
说明：样式名 ”adwraper“ 固定，id ”topad_xxx“ 为广告位标签
		xxx为传漾广告最后的数字
