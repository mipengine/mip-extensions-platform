# mip-swt

mip-swt 组件说明

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/extensions/platform/v1/mip-swt/mip-swt.js

## 示例

### 基本用法
```html
<mip-swt>
    自定义内容，可以嵌套其他组件
</mip-swt>
```

## 属性

### swt-time

说明：弹窗关闭后再次显示时间
必选项：否
类型：int
取值范围：0+
单位：ms
默认值：15000

### show-time

说明：打开页面显示弹窗停顿时间
必选项：否
类型：int
取值范围：0+
单位：ms
默认值：0
