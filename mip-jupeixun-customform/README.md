标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-jupeixun-customform/mip-jupeixun-customform.js

```html
<mip-jupeixun-customform>
    自定义内容，可以嵌套其他组件
</mip-jupeixun-customform>
```
