# mip-jt-resize

mip-jt-resize 金投适配

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-jt-resize/mip-jt-resize.js

## 示例

### 基本用法
```html
<mip-jt-resize width="640"></mip-jt-resize>
```

## width

### 适配的屏幕宽度

说明：适配的屏幕宽度  默认750
必选项：否
类型：int
取值范围：640 750 ..
单位：无
默认值：无
