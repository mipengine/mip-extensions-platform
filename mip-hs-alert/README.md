# mip-hs-alert

mip-hs-alert 组件说明
点击弹出对应的弹框显示，并阻止默认事件，点击页面后隐藏弹框。
标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-hs-alert/mip-hs-alert.js

## 示例

### 基本用法
```html
<mip-hs-alert>
    <button class="topbtn"></button>
	<div class="showself"></div>
</mip-hs-alert>
```
