----|----

## 示例

### 点击关闭广告
示例如下:

```html
   
   <mip-close-ad ></mip-close-ad>
```

### 基本用法
```html
<mip-close-ad>
  里面是点击关闭的按钮
</mip-close-ad>
```
说明：<mip-close-ad></mip-close-ad> 点击这个组件可以关闭其父级元素
