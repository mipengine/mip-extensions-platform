# mip-zol-ad

mip-zol-ad 是zol广告组件。

标题|内容
----|----
类型|业务,广告
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/extensions/platform/v1/mip-zol-ad/mip-zol-ad.js

## 示例

### banner
```html
<mip-zol-ad pid="26203"></mip-zol-ad>
```

### 嵌入代码
```html
<mip-zol-ad pid="5154"></mip-zol-ad>
```

## 属性

### pid
说明：广告id
必选项：是
类型：数字
取值范围：>0
