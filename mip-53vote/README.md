# mip-53vote 投票组件

mip-53vote 用来进行投票的小组件

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/extensions/platform/v1/mip-53vote/mip-53vote.js

##  示例

只需要一个`<mip-53vote></mip-53vote>`标签，

```html
<mip-53vote tid="254982">
	<div>点我一下你试试！</div>
</mip-53vote>
```


## 属性

### mip-53vote

说明：用来进行投票的小组件
必选项：否
